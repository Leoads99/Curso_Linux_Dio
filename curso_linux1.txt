nome_usuario@ubuntu:~$

O til significa que estamos em nossa pasta pessoal

pwd = lugar onde nos encontramos 

ls = lista os arquivos, lista
os diretórios existentes do caminho onde estiver

ls nome_pasta = retorna um caminho de uma pasta específica 
por exemplo: ls Documentos 

dir = também lista, compatibilidade com cmd 

cd nome_pasta = muda de diretório, 'change director'

mkdir = cria um diretório, ou pasta

rmdir nome_diretorio

ou rm -r = remove um diretório

rm nome_arquivo.txt = apaga arquivo

por exemplo: mkdir teste 

cd .. = retorna uma pasta

cd / = retorna para a pasta raiz '/'

ls -l = listar diretórios com detalhes 

man ls = retorna o manual do ls

ls --help = retorna o manual em português

cd ~ = volta para a pasta pessoal

history = histórico de comandos usados no terminal

!! = executa o último comando digitado

mv nome_pasta novo_nome = renomeia pasta

mv novo_nome ~ = move pasta para pasta raiz

mv nome_arquivo.txt /home/kira//Downloads

touch nome_arquivo.txt = cria arquivos vazios sem conteúdo

cp nome_arquivo /home/kira = copia arquivo 

tab auto completa


ATALHOS : 

!! = repete o último comando 

Ctrl+C  = cancela o comando atual em funcionamento 

Ctrl+Z  = pausa o comando atual, em primeiro plano ou segundo plano

Ctrl+D  = faz logout da sessão atual

Ctrl+W  = apaga uma palavra na linha atual

Ctrl+U  = apaga a linha inteira

Ctrl+R  = busca um comando recente


exit = sair do terminal
